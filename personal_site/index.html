About this site

In one of my courses, I learned about HTML and CSS, while culminated in basic site making. I decided to use this new skill to display my knowledge and experience in other areas as well!

The initial format was laid out using Boostratpr.io, which was the tool we were taught about in my Principles of Computing course. From there, I worked primarily in Visual Studio Code before switching to Sublime Text editor, and consulted Stack Overflow often to resolve any issues.

From there, I used Github LFS in order to publish my site to Github, due to some of the larger files. There was a lot of troubleshooting and some terminal commands, but eventually I was able to get it published. This is the first ever site I've created on my own, and though it is imperfect, I am very proud of it!
